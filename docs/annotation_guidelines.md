# Guía de anotación visual e iconográfica

## Propósito

Este documento define criterios simples y controlados para anotar atributos visuales en la muestra del corpus piloto de textiles andinos.

La anotación no busca descifrar el significado cultural definitivo de los motivos. Su objetivo es registrar rasgos visuales observables que puedan ser usados en experimentos computacionales posteriores.
