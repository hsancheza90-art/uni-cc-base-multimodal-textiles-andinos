# Ethical Use

Este proyecto trabaja con objetos patrimoniales andinos digitalizados por instituciones museográficas.

## Principios

1. No asumir que la disponibilidad digital equivale a apropiación libre.
2. Registrar siempre la fuente original.
3. Respetar las licencias de cada institución.
4. No eliminar marcas de agua ni alterar atribuciones institucionales.
5. No presentar inferencias algorítmicas como verdades culturales absolutas.
6. Evitar discursos de desciframiento total o apropiación interpretativa.
7. Reconocer los límites de los metadatos curatoriales.
8. Considerar que los textiles pueden tener valor cultural, espiritual, histórico o comunitario.

## Uso previsto

El dataset se construye para investigación académica en ciencias computacionales, aprendizaje multimodal y análisis de objetos culturales.

## Uso no previsto

No se recomienda usar este dataset para:

- Comercialización directa de diseños patrimoniales.
- Reproducción no autorizada de imágenes protegidas.
- Clasificación cultural automática sin validación experta.
- Afirmaciones concluyentes sobre significado simbólico sin respaldo antropológico.
